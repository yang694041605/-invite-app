# 共振分析模块使用指南

## 概述

共振分析模块是数字分析系统的核心模块之一，位于"数据输入"之后，"智能预测"之前。该模块基于多周期数据分析，通过识别多种模式，为后续预测提供参考。

## 功能特点

### 1. 数据接收与状态显示

- 接收数据输入页面传递的历史数据
- 显示导入数据的起始期数、终止期数和预测期数
- 提供数据状态可视化展示

### 2. 多周期数据分析

- 5期分析：捕捉即时趋势
- 10期分析：确认趋势稳定性
- 20期分析：概率回归压力

### 3. 三维权重模型

- 基于5期、10期、20期数据构建权重模型
- 多维度评估号码出现概率

### 4. 模式识别

- 识别多种数据模式，提供不同类型的号码推荐
- 每种模式都有相应的策略建议

### 5. 号码推荐

- 提供多种号码组合方案
- 包括核心号码、五码组合和七码组合
- 覆盖不同概率范围的组合

### 6. 跨度和形态分析

- 分析历史跨度分布，预测可能的跨度范围
- 评估不同形态的出现概率

### 7. 组合方案生成

- 根据分析结果生成多种组合方案
- 提供不同类型的组合供选择

## 使用流程

1. **数据准备**：在数据输入页面导入或输入历史数据
2. **进入分析**：点击"共振分析"按钮进入分析页面
3. **开始分析**：点击"开始分析"按钮，系统将自动进行多周期数据分析
4. **查看结果**：分析完成后，可以在各个标签页查看详细分析结果
5. **前往预测**：点击"前往智能预测"按钮，使用分析结果进行最终预测

## 技术实现

### 文件结构

```
pages/
├── resonance.html      # 共振分析页面
├── data-input.html     # 数据输入页面
└── prediction.html     # 智能预测页面
```

### 数据流转

1. **数据输入页面** → 收集历史数据 → 保存到本地存储
2. **共振分析页面** → 从本地存储读取数据 → 进行分析 → 保存结果
3. **智能预测页面** → 从本地存储读取分析结果 → 生成最终预测

## 注意事项

1. **数据质量**：建议至少提供20期历史数据以保证分析准确性
2. **结果参考**：分析结果仅供参考，实际使用需结合其他因素综合考虑
3. **定期更新**：建议定期更新历史数据以保持分析模型的准确性
